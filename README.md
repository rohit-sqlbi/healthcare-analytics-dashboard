# Healthcare Analytics Dashboard

## Project Overview

This project is an end-to-end Healthcare Analytics Dashboard built using Power BI, SQL, and DAX. The dashboard provides insights into hospital operations, patient admissions, doctor performance, revenue analysis, treatment analytics, and emergency case monitoring.

---

## Tools & Technologies Used

* Power BI
* SQL
* DAX


---

## Dashboard Pages

### 1. Executive Dashboard

* Total Revenue
* Total Patients

### 2. Doctor & Department Analytics

* Department-wise Revenue
* Doctor Performance Analysis

* Payment Status Analysis

* Detailed patient-level and department-level analysis using drill-through reports.

---

## Key Business Insights

1. The Cardiology department generated the highest revenue compared to other departments.

2. Most treatments were completed successfully, while cancelled treatments were comparatively low.

3. Monthly trend analysis showed that patient admissions were higher during certain months.

4. Top-performing doctors significantly contributed to overall hospital revenue.

5. The average doctor rating remained at a high level, indicating strong patient satisfaction and service quality.

6. The number of completed appointments was significantly higher than cancelled appointments.

7. Time-series analysis can help estimate future patient load and revenue trends.
